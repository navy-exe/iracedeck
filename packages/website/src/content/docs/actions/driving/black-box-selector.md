The Black Box Selector action lets you jump directly to any of iRacing's black box overlays or cycle through them sequentially. Key bindings are shared globally across all instances of this action.

## Modes

### Direct Selection

| Mode | Description |
|------|-------------|
| Lap Timing | Opens the Lap Timing black box. |
| Standings | Opens the Standings black box. |
| Relative | Opens the Relative black box. |
| Fuel | Opens the Fuel black box. |
| Tires | Opens the Tires black box. |
| Tire Info | Opens the Tire Info black box. |
| Pit Stop | Opens the Pit Stop black box. |
| In-Car | Opens the In-Car black box. |
| Mirror/Graphics | Opens the Mirror/Graphics black box. |
| Radio | Opens the Radio black box. |
| Weather | Opens the Weather black box. |

### Cycle

| Mode | Description |
|------|-------------|
| Next | Cycles to the next black box. |
| Previous | Cycles to the previous black box. |

## Encoder Support

Yes — rotation cycles through the black boxes (clockwise for next, counter-clockwise for previous).

:::note
This action uses global key bindings shared across all instances. Configuring a key binding on one instance updates it for all Black Box Selector actions.
:::
