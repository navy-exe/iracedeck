iRaceDeck provides a wide range of actions for controlling iRacing from your Stream Deck. Actions are organized into the following categories:

## Categories

| Category | Actions | Description |
|----------|---------|-------------|
| [Display & Session](/actions/display-session/session-info/) | 2 | Live telemetry display and session information |
| [Driving](/actions/driving/black-box-selector/) | 4 | Black box, look direction, car control, and audio |
| [Cockpit](/actions/cockpit/cockpit-misc/) | 4 | Dashboard controls, splits, telemetry logging, and UI toggles |
| [Pit Service](/actions/pit-service/) | Pit stop commands and fuel management |
| [View & Camera](/actions/view-camera/) | Camera controls and replay |
| [Communication](/actions/communication/) | Chat macros and radio |
| [Car Setup](/actions/car-setup/) | Setup adjustments |
| [Media](/actions/media/) | Screenshot and video capture |

## How Actions Work

Most actions support multiple **modes** selectable via the Property Inspector. For example, the Session Info action can display incidents, time remaining, lap count, position, fuel level, or active flags — all from a single action type.

Actions that support **encoder** can also be used with the Stream Deck+ dial for rotation-based input (e.g., cycling through black boxes or adjusting volume).

All keyboard shortcuts used by actions are **user-configurable** through the Property Inspector to match your iRacing key bindings.
