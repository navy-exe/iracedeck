# Trigger Windshield Wipers

Triggers a single wipe of the windshield wipers.

## Properties

| Property | Value |
|----------|-------|
| Action ID | `com.iracedeck.sd.core.trigger-windshield-wipers` |
| Type | Button |
| SDK Support | No |
| Encoder Support | No |

## Behavior

### Button Press
- Performs a single wiper sweep

## Settings

None.

## Keyboard Simulation

| Action | Key | iRacing Setting |
|--------|-----|-----------------|
| Trigger | Ctrl+Alt+W | Trigger Windshield Wipers |

## Icon States

| State | Description |
|-------|-------------|
| Default | Wiper icon |

## Notes

- Single sweep action, not continuous
- For continuous wipers, use "Toggle Windshield Wipers" binding in iRacing
- Essential for wet weather racing visibility
