# Toggle Display Reference Car

Toggles the reference car display for split-time comparison.

## Properties

| Property | Value |
|----------|-------|
| Action ID | `com.iracedeck.sd.core.toggle-display-reference-car` |
| Type | Toggle |
| SDK Support | No |
| Encoder Support | No |

## Behavior

### Button Press
- Toggles the reference car display on/off

## Settings

None.

## Keyboard Simulation

| Action | Key | iRacing Setting |
|--------|-----|-----------------|
| Toggle | Ctrl+C | Toggle Display Reference Car |

## Icon States

| State | Description |
|-------|-------------|
| Off | Reference car display disabled |
| On | Reference car display enabled |

## Notes

- Shows which car you're comparing split times against
- Useful for tracking specific competitors during a race
