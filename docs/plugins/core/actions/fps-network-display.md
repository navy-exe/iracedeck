# FPS/Network Display

Toggles the FPS and network statistics overlay.

## Properties

| Property | Value |
|----------|-------|
| Action ID | `com.iracedeck.sd.core.fps-network-display` |
| Type | Toggle |
| SDK Support | No |
| Encoder Support | No |

## Behavior

### Button Press
- Toggles the display on/off

## Settings

None.

## Keyboard Simulation

| Action | Default Key | iRacing Setting |
|--------|-------------|-----------------|
| Toggle | F | FPS / Network Display |

## Icon States

| State | Description |
|-------|-------------|
| Off | Display hidden |
| On | Display visible |

## Notes

- Shows frames per second and network latency/quality
- Useful for diagnosing performance issues
- Can be positioned via UI Edit mode
