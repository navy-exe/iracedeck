# Toggle Weather Radar

Toggles the weather radar overlay display.

## Properties

| Property | Value |
|----------|-------|
| Action ID | `com.iracedeck.sd.core.toggle-weather-radar` |
| Type | Toggle |
| SDK Support | No |
| Encoder Support | No |

## Behavior

### Button Press
- Toggles the weather radar on/off

## Settings

None.

## Keyboard Simulation

| Action | Key | iRacing Setting |
|--------|-----|-----------------|
| Toggle | Shift+Alt+R | Toggle Weather Radar |

## Icon States

| State | Description |
|-------|-------------|
| Off | Radar hidden |
| On | Radar visible |

## Notes

- Shows precipitation and weather patterns on track map
- Essential for dynamic weather races
- Helps plan pit strategy around incoming rain
