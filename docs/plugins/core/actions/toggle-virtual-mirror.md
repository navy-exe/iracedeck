# Toggle Virtual Mirror

Toggles the virtual rear-view mirror display.

## Properties

| Property | Value |
|----------|-------|
| Action ID | `com.iracedeck.sd.core.toggle-virtual-mirror` |
| Type | Toggle |
| SDK Support | No |
| Encoder Support | No |

## Behavior

### Button Press
- Toggles the virtual mirror on/off

## Settings

None.

## Keyboard Simulation

| Action | Key | iRacing Setting |
|--------|-----|-----------------|
| Toggle | Alt+M | Toggle Virtual Mirror |

## Icon States

| State | Description |
|-------|-------------|
| Off | Mirror hidden |
| On | Mirror visible |

## Notes

- Shows a floating rear-view mirror on screen
- Useful for cars without adequate in-car mirrors
- Position and size adjustable in iRacing settings
