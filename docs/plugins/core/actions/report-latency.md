# Report Latency

Reports current network latency to the chat.

## Properties

| Property | Value |
|----------|-------|
| Action ID | `com.iracedeck.sd.core.report-latency` |
| Type | Button |
| SDK Support | No |
| Encoder Support | No |

## Behavior

### Button Press
- Reports latency information to session chat

## Settings

None.

## Keyboard Simulation

| Action | Key | iRacing Setting |
|--------|-----|-----------------|
| Report | L | Report Latency |

## Icon States

| State | Description |
|-------|-------------|
| Default | Latency/network icon |

## Notes

- Posts your current ping/latency to the session chat
- Useful for diagnosing connection issues during races
- Visible to other drivers in the session
