# Speed/Gear/Pedals Display

Toggles the speed, gear, and pedals overlay display.

## Properties

| Property | Value |
|----------|-------|
| Action ID | `com.iracedeck.sd.core.speed-gear-pedals-display` |
| Type | Toggle |
| SDK Support | No |
| Encoder Support | No |

## Behavior

### Button Press
- Toggles the display on/off

## Settings

None.

## Keyboard Simulation

| Action | Default Key | iRacing Setting |
|--------|-------------|-----------------|
| Toggle | P | Speed / Gear / Pedals Display |

## Icon States

| State | Description |
|-------|-------------|
| Off | Display hidden |
| On | Display visible |

## Notes

- Shows current speed, selected gear, and pedal inputs
- Useful for debugging input issues or learning
- Can be positioned via UI Edit mode
