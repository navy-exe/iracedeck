# Recenter VR View

Recenters the virtual reality headset view.

## Properties

| Property | Value |
|----------|-------|
| Action ID | `com.iracedeck.sd.core.recenter-vr-view` |
| Type | Button |
| SDK Support | No |
| Encoder Support | No |

## Behavior

### Button Press
- Recenters the VR view to current head position

## Settings

None.

## Keyboard Simulation

| Action | Key | iRacing Setting |
|--------|-----|-----------------|
| Recenter | ; | Recenter VR View |

## Icon States

| State | Description |
|-------|-------------|
| Default | VR recenter icon |

## Notes

- Essential for VR users when view drifts
- Sets current head position as the new center point
- Should be used when seated in normal driving position
