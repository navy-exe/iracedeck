# Autofuel Toggle

Toggles the automatic fuel calculation feature.

## Properties

| Property | Value |
|----------|-------|
| Action ID | `com.iracedeck.sd.core.autofuel-toggle` |
| Type | Toggle |
| SDK Support | No |
| Encoder Support | No |

## Behavior

### Button Press
- Toggles autofuel on/off

## Settings

None.

## Keyboard Simulation

| Action | Key | iRacing Setting |
|--------|-----|-----------------|
| Toggle | Shift+Ctrl+A | Autofuel Toggle |

## Icon States

| State | Description |
|-------|-------------|
| Off | Autofuel disabled |
| On | Autofuel enabled |

## Notes

- When enabled, iRacing automatically calculates fuel needed for remaining laps
- Works with the Autofuel Lap Margin setting
- Useful for endurance races to ensure adequate fuel
