# Toggle Dash Box

Toggles the in-car dash information box display.

## Properties

| Property | Value |
|----------|-------|
| Action ID | `com.iracedeck.sd.core.toggle-dash-box` |
| Type | Toggle |
| SDK Support | No |
| Encoder Support | No |

## Behavior

### Button Press
- Toggles dash box visibility on/off

## Settings

None.

## Keyboard Simulation

| Action | Key | iRacing Setting |
|--------|-----|-----------------|
| Toggle | D | Toggle Dash Box |

## Icon States

| State | Description |
|-------|-------------|
| Off | Dash box hidden |
| On | Dash box visible |

## Notes

- Shows additional information overlay on the dashboard
- Useful for accessing quick info without opening black boxes
