# Toggle UI Edit

Toggles UI edit mode for repositioning on-screen elements.

## Properties

| Property | Value |
|----------|-------|
| Action ID | `com.iracedeck.sd.core.toggle-ui-edit` |
| Type | Toggle |
| SDK Support | No |
| Encoder Support | No |

## Behavior

### Button Press
- Toggles UI edit mode on/off

## Settings

None.

## Keyboard Simulation

| Action | Default Key | iRacing Setting |
|--------|-------------|-----------------|
| Toggle | Alt+K | Toggle UI Edit |

## Icon States

| State | Description |
|-------|-------------|
| Off | Normal mode |
| On | Edit mode active |

## Notes

- When enabled, UI elements can be dragged to reposition
- Useful for customizing HUD layout
- Exit edit mode to lock positions
